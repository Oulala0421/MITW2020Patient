Active

是否為使用中的病人。
1.數值個數(0-…1) ，可不提供，或給定 true 或 false。
2.此欄位可標示長期未到該單位就醫的病人，或者依據醫療單位的商業規則定義之，
主要用以篩選病人清單排除不常見/不活躍(inactive)的病人。
3.基於某些理由，「過逝的，病患人」死亡也可能被設為inactive(意即active value=false)，
但也可能病人過逝後一段時間，也可將又被設定為 active 值設為 false(意即active value=true)。

議題：

可否用來標示照護中心或特定醫院已出院之病人?

格式如下:

                {
                    "active":"true"
                }
