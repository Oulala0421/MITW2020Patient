Telecom

病人聯絡資訊。數值個數(0…-1)，可不提供，或記錄電話、手機號碼、email、社交軟體帳號(如 Line、WeChat、Facebook) 等多組聯絡資訊。

議題：
1.電話號碼是否分段，如0954787878 改為0954 787 878 or 0954-787-878
2.是否有國際聯絡電話需求，其建議格視為何？
或參考https://www.itu.int/rec/T-REC-E.123-200102-I/e
3.簡訊自動通知有其需求，自動發送簡訊機制需考量。是否可設定群組通知，其群組 ID 為何？
是否需考慮社群群組及簡訊自動通知權限管控機制？

各欄位細部規格如下:

  • system
    (通訊系統)

  • value
    (對應之帳號或號碼)

  • use
    (此帳號或號碼用途)

  • rank
    (聯絡資訊優先次序)

  • period
    (使用期限)
