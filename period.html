period

使用期限。
產生次數(0…-1)；通常不確定聯絡資訊使用期限，因此不包含此資訊(0)。若清楚聯絡資訊使用期限，則提供開始及結束時間。

格式如下:

                {
                    telecom:
                        {
                            "period": 
                                {
                                    "end": "2019"
                                }
                        }
                }
