system

通訊系統。
1.數值個數(0-1)，可不提供或數值可為phone | fax | email | pager | url | sms | other 等
2.phone, fax, pager, and email 可直接作為 system 的 value 屬性
3.url 及 other 則以對應之通訊系統連結如 FB之連結https://www.facebook.com/ 作為此 system 的值

格式如下:

                    {
                    telecom:
                    {
                    "system" :"phone"
                    }
                    }
