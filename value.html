value

通訊系統中對應之帳號或號碼。
1.數值個數(0-1)，可能沒有或不提供，也可提供電話號碼、email 帳號、健康平台病人線上聯絡帳號、FB 帳號等。
2.電話、手機、簡訊格式建議存放地區(National) 可直接撥接之標準規格
如台灣地區之家裡電話：: (03)8561234公司電話 (04) 22513333 ext 5438，手機號碼 0954787878 等

格式如下:

                {
                    telecom:
                        {
                            "value":"(03) 5555 6473"
                        }
                }
